## Run the project
1. Clone repository

    ```
        1.1- git clone https://github.com/AhmedAliSayed13/task-uaebarq.git
        1.2- cd task-uaebarq 
        1.3- composer install
        1.4- cp .env.example .env
        1.5- php artisan key:generate
        1.6- php artisan serve
    ```
